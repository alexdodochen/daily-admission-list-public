## 步驟一：匯入病人資料

### 輸入
使用者貼入院名單截圖（可多天），格式範例：「這是 20260412-13 的入院清單」

### 流程

1. **OCR 辨識**截圖中的病人資料（12 欄 A-L）：
   實際住院日 | 開刀日 | 科別 | 主治醫師 | 主診斷 | 姓名 | 性別 | 年齡 | 病歷號碼 | 病床號 | 入院提示 | 住急

2. **主治醫師名單校正**（OCR 常誤讀）：
   正確名單：陳柏升、許志新、詹世鴻、李柏增、陳昭佑、廖瑀、黃鼎鈞、陳柏偉、劉嚴文、陳儒逸、鄭朝允、劉秉彥、陳則瑋、張獻元、黃睦翔、林佳凌、張倉惟、王思翰、柯呈諭、李文煌
   常見修正：鄭亦汝→鄭朝允、黃曉翊→黃睦翔、柯星諭→柯呈諭

3. **建立 Python 腳本** `create_sheet_YYYYMMDD.py`，參考 `create_sheet_20260412.py` 範本：
   - 用 `create_worksheet()` 建立日期工作表
   - 寫入 A-L 標題和病人資料
   - 寫入 N-U 入院序列標題（8欄：序號|主治醫師|病人姓名|備註|病歷號|術前診斷|預計心導管|每日續等清單）
   - 按主治醫師分組，在主資料下方用 `write_doctor_table()` 建立醫師病人表格（8欄 A-H）
   - 設定欄寬、格式、邊框

4. **執行腳本**，列出所有病歷號請使用者確認（OCR 數字容易出錯）

5. **病歷號確認後，立即請使用者提供 EMR session URL**
   - 不需額外指令，直接說：「病歷號確認，請登入 EMR 後貼 session URL 給我」
   - 拿到 URL 後自動批次查詢所有病人 EMR
   - EMR 摘要自動寫入 Sheet C/D 欄
   - 無資料病人自動標記 H 欄 + 寫入無資料病人工作表
   - 用 EMR 正確姓名更新 Sheet 上的 OCR 錯誤名字
   - **整個流程不需使用者再下任何指令**

### 重要規則
- 病歷號固定 8 碼，以文字存入（保留前導零）
- 截斷的姓名不需補全，EMR 步驟會自動校正
- 寫入前先確認目標區域為空，不覆蓋現有資料
- 匯入 → 確認 → EMR 是一條龍流程，中間不停頓
