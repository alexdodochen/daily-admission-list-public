name: 抽籤與 Round-Robin 正確邏輯
description: 抽籤決定醫師順序，入院序用真正的round-robin（輪流一人一人），非時段醫師最後才加入
type: feedback
---

**Round-Robin 規則（之前實作錯誤，已修正）：**
- 抽籤決定醫師順序（如 A→B→C）
- 入院序：A第1人 → B第1人 → C第1人 → A第2人 → B第2人 → C第2人...
- **不是** A全部 → B全部 → C全部（那是錯的）
- 各醫師內部順序依 E 欄（手動設定入院序）決定，無E欄的按原始順序

**抽籤流程：**
1. 先處理手動指定的強制優先（直接給#1，不進rotation）
2. 手動指定醫師順位（如「詹世鴻第1、黃睦翔第2」）
3. 剩餘時段醫師隨機抽籤（*2 醫師兩支籤）
4. 主 round-robin 跑完所有時段醫師病人
5. **非時段醫師**（不在隔日抽籤表）等全部時段醫師排完後，問是否與**每日續等清單**合併抽籤

**Why:** 之前錯把每位醫師的病人連排（block），應為輪流一人一人的round-robin。

**How to apply:** 抽籤時建立每位醫師的 patient queue，再用 while any(queues) 輪流取各醫師第一位病人。
