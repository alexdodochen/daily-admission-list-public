---
name: 星期五入院導管排同日
description: 星期五入院的病人，cathlab 排在同一天星期五（不是星期六），因為週六無排程
type: feedback
---

星期五入院需要做心導管的病人，排程日 = 入院日（同一天星期五），**不是 N+1 的週六**。

**Why:** 週六沒有正常導管排程／抽籤表，所以週五入院的病人 cathlab 必須塞回當天週五。這和其他日子「入院日 N → 導管日 N+1」的規則不同。之前已有「五→五」抽籤規則，本條進一步確認 cathlab 實際執行日也是同一天。

**How to apply:**
- 入院日為星期五 → cathlab 日也為同一天星期五
- 時段查主治醫師時段表的週五欄
- `verify_cathlab.py` 對週五入院時需特別處理（不是 +1 天）
- 其他天照舊：日→一、一→二、二→三、三→四、四→五
