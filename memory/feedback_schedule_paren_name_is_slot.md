name: 主治醫師時段表括號內名字 = 該日有時段
description: 讀時段表時，括號裡單獨出現的醫師名（如「(陳則瑋)」）也算當天該時段有班，不是純註記。
type: feedback
---

讀「主治醫師導管時段表」時，若某日某時段格子下方另一列出現括號內醫師名（例如週二下午 H1：劉秉彥 / (陳則瑋)），表示該醫師當天**也有時段**，不是備註或副刀。

**Why:** 4/21（Tue）排程時，schedule_dump.json 顯示週二 PM H1 為 "劉秉彥" + 下一列 "(陳則瑋)"。我把 (陳則瑋) 當成註記，將其兩位病人（沈維護/戴宗訓）誤排為「本日無時段」H1 2100/2101。實際上週二 PM H1 同時掛 劉秉彥 與 陳則瑋 兩位，陳則瑋的病人應排在 H1 1800+ 正常 PM 時段。

**How to apply:**
- 解析 schedule_dump.json / 主治醫師導管時段表時，把同欄連續多列的格子合併視為「該時段多位醫師」。
- 括號區分用途：
  - `醫師A(寬)`、`醫師A(浩)`、`醫師A(齡)` 等單字 → A 是主刀，括號內是第二主刀代號
  - `(醫師A)`（整個格子只有括號名）→ 該醫師也是當天此時段的有時段醫師（與正上方那位並列）
- 排 cathlab 時，這類「括號獨立列」的醫師按正常時段排（AM 0600+ / PM 1800+），不要丟到 2100+「本日無時段」。

已知案例：
- 週二 PM H1：劉秉彥 + (陳則瑋)
