---
name: 張獻元週二病人依註記決定同日或隔日
description: 張獻元週二入院病人若註記無王思翰/張倉惟 → 週二PM同日；有標記 → 週三張獻元時段做
type: feedback
---

**規則：** 張獻元主治、**週二入院** 的病人：
- 子表格 H 欄（註記）**沒有**「王思翰」或「張倉惟」→ **週二 PM 同日** 做，用張獻元週二自己的時段
- 子表格 H 欄**有**「王思翰」或「張倉惟」→ **週三（N+1）做**，用張獻元週三的導管時段（不是王思翰/張倉惟 自己的時段，他們本身沒有固定時段，是借張獻元的）

**實例（4/14 週二）：**
- 黃郭玉霞（PAOD/PTA）註記「排4/14PM做」→ 4/14 PM 張獻元時段 ✓
- 陳許黃焿（PAOD/PTA）註記「12C已給床」→ 4/14 PM 張獻元時段 ✓
- 王樹英（PAOD/PTA）註記「張倉惟 腳已黑」→ 4/15 張獻元時段
- 黃鉦傑（CAD）註記「王思翰 希望AM做」→ 4/15 張獻元時段
- 蔡志祥（CAD/PCI）註記「張倉惟」→ 4/15 張獻元時段

**Why:** 張獻元自己有週二 PM 的 PAD/PTA 時段，病人入院同日做是常規。王思翰、張倉惟 沒有獨立時段，借張獻元的時段 → 他們的病人 admission 仍是週二，但排到週三 張獻元的週三時段做 cath（遵守 N→N+1）。

**How to apply:**
1. 做 cathlab keyin 或 verify 時先看 H 欄註記判斷是同日 PM 還是 N+1
2. 無標記 → cathlab_date = 入院日當天；有標記 → cathlab_date = 入院日+1 且以張獻元當天時段 key
3. 不論哪種都 key 在「attendingdoctor=張獻元」，只是時段/日期不同
