---
name: 張獻元病人預設週二同日下午做
description: 張獻元病人週二入院若備註無「王思翰」或「張倉惟」→ 週二下午用張獻元自己時段做導管
type: feedback
---

**規則：** 張獻元主治的病人，**週二入院** 時：
- 若子表格 H 欄（註記）**沒有** 「王思翰」或「張倉惟」字樣
  → **週二下午** 做導管（同日），使用張獻元自己的時段
- 若註記含「王思翰」→ 照王思翰時段排（通常 4/15 AM）
- 若註記含「張倉惟」→ 照張倉惟時段排

**實例（4/14 週二）：**
- 黃郭玉霞（PAOD/PTA）註記「排4/14PM做」→ 4/14 PM 張獻元時段 ✓
- 陳許黃焿（PAOD/PTA）註記「12C已給床」→ 4/14 PM 張獻元時段 ✓
- 王樹英（PAOD/PTA）註記「張倉惟 腳已黑」→ 跟張倉惟時段 ✗（非同日）
- 黃鉦傑（CAD）註記「王思翰 希望AM做」→ 跟王思翰時段 ✗（非同日）
- 蔡志祥（CAD/PCI）註記「張倉惟」→ 跟張倉惟時段 ✗（非同日）

**Why:** 張獻元自己有週二下午 PAD/PTA 的時段，入院當天同日做導管是他的常規。只有當病人被指定給 王思翰（週三 AM 主導）或 張倉惟（另一個時段）時才會跨日。

**How to apply:**
1. 做 cathlab keyin 或 verify 時：張獻元的病人若週二入院且註記無王思翰/張倉惟 → cathlab 日期 = 入院日（同日，非 N+1）
2. verify_cathlab.py 預期這些病人出現在 **入院日當天** 的 WEBCVIS，不在 N+1
3. 住院序也應反映：這些病人的「開刀日」可能 = 「實際住院日」（同日）
