# 各日時段醫師清單（Lottery 用）

**權威來源**：Google Sheet 工作表 `主治醫師抽籤表`，col A-E = 週一-五，`*2` 後綴 = 抽兩支籤（兩倍機率）。

**使用規則**：
- 抽籤 / 排入院序時，醫師必須出現在當日該欄 → 時段醫師
- 不在當日欄 → **非時段醫師**（lottery 排時段後，cathlab H1 2100+ 註記「本日無時段」）
- **特例**：詹世鴻 Friday 雖在 col E，**視為非時段**（CLAUDE.md rule #16，memory `feedback_zhan_friday_pm.md`）
- **入院日→cathlab日**：日→一、一→二、二→三、三→四、四→五、**五→五（同日）**

若需查 cathlab 房間（H1/H2/C1/C2）+ 時段（AM/PM），讀 `主治醫師導管時段表`（或 `schedule_readable.txt`）。**不要拿該表當 lottery 依據**，兩表用途不同。

## 維護

抽籤表會隨人事異動（新醫師、留職停薪、離職）更新。若使用者說某醫師「最近不抽」「改成 *2」等，就回 Google Sheet 讀 latest，並更新此 memory。
