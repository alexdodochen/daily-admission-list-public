---
name: 星期五入院用星期五抽籤表
description: 星期五入院的病人，抽籤和導管時段查的是星期五的抽籤表，不是星期六
type: feedback
---

**規則：** 星期五入院的病人，抽籤排序要查的是**星期五的導管時段表和抽籤表**。

入院日 → 查的抽籤表/時段表：
- 週日入院 → 查週一
- 週一入院 → 查週二
- 週二入院 → 查週三
- 週三入院 → 查週四
- 週四入院 → 查週五
- **週五入院 → 查週五**（不是週六，週六沒有抽籤表）

**Why:** 週六沒有標準抽籤表。週五入院的病人導管通常排在週五下午或隔週一，所以用週五時段表判斷。
**How to apply:** 判斷入院日星期幾時，若為週五則不 +1，直接用週五的抽籤表。
