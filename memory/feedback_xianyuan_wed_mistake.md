---
name: 張獻元週三入院多半是王思翰/張倉惟 key 錯
description: 週三若看到張獻元預約入院，先懷疑是王思翰或張倉惟借時段但住院日寫錯（應為週二入院）
type: feedback
---

**規則：** 王思翰、張倉惟會借張獻元的導管時段，他們的病人應該是**週二入院、週三做 cath**。若看到張獻元名下的病人標記「週三入院」，高機率是 keyin 時把住院日輸錯（寫成 cath 日）。

**Why:** 王思翰/張倉惟本身沒有固定 cath 時段，借張獻元的時段時 admission → cath 仍遵守 N → N+1 規則。OPD 系統裡住院醫師可能只選「張獻元」當主治，導致住院日被填成週三（cath 當天）而非週二。

**How to apply:**
- 週三入院清單出現「張獻元」時，先檢查該病人實際主治是否為王思翰/張倉惟
- 若是 → 提醒使用者很可能是 keyin 錯誤，住院日應改為週二
- 不要直接改，先問使用者確認
- 正常週三入院的張獻元本人病人仍可能存在（少數），不要一律當錯
