name: EMR完成後自動預填F/G欄
description: EMR摘要完成後，根據摘要內容自動預填術前診斷(F)和預計心導管(G)下拉選單，讓使用者檢查確認
type: feedback
---

**規則：** EMR 摘要寫入 C/D 欄後，Claude 根據摘要內容自動預填 F 欄（術前診斷）和 G 欄（預計心導管），然後列出讓使用者檢查。

**判斷邏輯：**
- 從摘要的「本次住院計畫」和「心臟科相關診斷」推斷 F/G 值
- F/G 值必須是下拉選單中的選項（來自「下拉選單」工作表 A 欄和 D 欄）
- 無法判斷時留空，標記讓使用者手動選

**流程變更：**
- 舊流程：EMR 完成 → 等使用者手動填 F/G → 使用者通知 Claude 整合
- 新流程：EMR 完成 → Claude 自動預填 F/G → 列出請使用者檢查 → 使用者確認後整合

**Why:** 減少使用者手動操作，大部分病人的 F/G 可以從 EMR 摘要直接判斷
**How to apply:** admission-emr-extraction skill 完成後，自動讀取下拉選單選項，比對摘要內容預填 F/G，再請使用者確認
