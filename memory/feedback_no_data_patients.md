---
name: 無資料病人自動處理
description: EMR查無紀錄時自動在H欄寫「無資料病人」並帶入無資料病人工作表，不需手動觸發
type: feedback
---

EMR 摘要步驟中，查無門診紀錄的病人自動執行以下三件事：

1. **C 欄**寫「無本院一年內主治醫師門診紀錄」
2. **H 欄（註記）**寫「無資料病人」
3. **自動帶入「無資料病人」工作表**（不需使用者另外下指令）

**目標工作表格式**（取代模式，每次先清空再寫入）：
| 預計入院日期 | 主治醫師 | 姓名 | 病歷號 | 備註 |

- 預計入院日期：從工作表名稱解析（20260407 → 2026/04/07）
- 備註：H 欄的完整內容
- 寫入方式：取代（每次清空後重寫，掃描當次 EMR 摘要涉及的所有工作表）

**Why:** 2026-04-08 使用者要求 EMR 步驟完成後自動整理無資料病人，不需額外下指令。

**How to apply:**
- EMR 批次完成後，掃描所有已處理工作表的醫師病人表格
- 找到 C 欄=「無本院一年內主治醫師門診紀錄」的病人
- 自動在 H 欄寫「無資料病人」
- 自動清空+重寫「無資料病人」工作表
