---
name: EMR姓名自動校正不需提醒
description: 讀到EMR上的姓名就應該自動更新Sheet所有位置的姓名，不需使用者提醒
type: feedback
---

EMR 擷取到病人姓名後，必須立即自動更新 Sheet 上所有位置：
- 醫師子表格 A 欄（姓名）
- 主資料區 G 欄（姓名）
- N-V 入院序列 P 欄（病人姓名）

**Why:** 2026-04-08 使用者明確指出：「你讀到EMR上的姓名就應該自動更新sheet上面的姓名，不要我提醒」。之前需要使用者確認截斷姓名才更新，太慢。

**How to apply:** EMR 步驟中，只要 EMR 回傳的姓名與 Sheet 上的不同（特別是有 "..." 截斷的），一律自動用 EMR 姓名覆寫，不問使用者。
