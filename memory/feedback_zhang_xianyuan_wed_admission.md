**規則：** 張獻元醫師的病人若出現在**週三入院名單**，視為**異常 case**，備註欄（子表格 H 欄）要明確標記「⚠ 疑王思翰/張倉惟誤key，應週二入院」。

**Why:** 張獻元自己的時段是 Tuesday PM（PAD/PTA）和其他固定時段。王思翰和張倉惟不在 Wed 抽籤表，但他們的病人需要借用 張獻元 的 Wed 時段做導管 → 正常流程是**週二入院、週三借張獻元時段做導管**（N+1 規則）。若行政誤把「開刀日」當「入院日」key 成週三，就會在週三入院名單看到 張獻元 病人，這是 key 錯資料不是實際住院。

**How to apply:**
1. 匯入每日入院清單後，若週三的主治醫師是 張獻元 → **標記備註** `⚠ 疑王思翰/張倉惟誤key，應週二入院`
2. 告知使用者這個 case 可能要改住院日
3. cathlab keyin 時仍依開刀日（通常是當週三）排，但 attendingdoctor 要確認是否應為 王思翰 或 張倉惟
4. 4/22 潘韋儒 (01580227) 是第一個發現的實例：週三入院、開刀日 4/22、主治 張獻元 → 很可能實際是 王思翰/張倉惟 週二入院 patient

**反例（正常 case）：**
- 週二入院、張獻元 主治 → 週二同日 PM 或週三做導管（正常 `feedback_zhang_xianyuan_same_day.md` 規則）
- 週四、週五入院、張獻元 主治 → 正常
